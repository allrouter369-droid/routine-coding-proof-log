# Sample Outputs

Use this directory for sanitized run records.

Do not include secrets, private code, customer data, or unredacted logs.

Recommended file name:

```text
YYYY-MM-DD-route-name-case-id.md
```

Each sample should include:

- route / endpoint;
- model name returned by the client;
- test case id;
- raw output or summarized output;
- pass / partial / fail;
- independent evidence used;
- human review notes.
